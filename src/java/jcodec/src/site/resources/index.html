Welcome to JCodec!

JCodec is an open source pure java implementation of video and audio codecs and formats.

Features Roadmap
H264 ( AVC ) Baseline profile decoder AAC decoder
Apple ProRes decoder and encoder H264 Main profile decoder
MJPEG decoder H264 Baseline profile encoder
Matroska ( MKV ) demuxer
MP4 ( MOV ) demuxer/muxer and tools
